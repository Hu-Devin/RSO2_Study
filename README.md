2. Ryubing —— Ryujinx 的社区接力版
项目背景：在 Ryujinx 官方宣布关停后，由社区开发者自发维护并接力开发的 Ryujinx 分支项目。
特点：
高还原度与准确性：继承了 Ryujinx 极高的游戏兼容性和画面还原度，非常适合游玩大作。
持续优化：持续改进了 Vulkan 性能、着色器编译（Shader Compilation）以及 QoL（易用性）体验。
联机与多人支持：保留并优化了本地联机（LDN）和联机大厅功能。
Linux 支持：提供原生的 Linux 构建版本。
3. Eden —— 性能与跨平台新星
项目背景：同样是基于 Yuzu 源码开发的高性能实验性模拟器，在 Android、PC/Linux 端都非常活跃。
特点：
一键迁移：首次启动时，支持自动检测并一键导入你之前在 Sudachi、Suyu 或 Yuzu 中留下的存档和数据，无缝切换。
实验性性能优化：拥有名为 "Eden's Valley" 的实验性设置面板，可为高端硬件提供更极限的帧率和画质表现。
界面清爽，对 Linux 系统的适配和手柄适配非常优秀。
